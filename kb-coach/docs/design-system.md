# KB教练 — 设计规范

## 品牌

- **名称**：KB教练
- **Slogan**：拍张照，练出型
- **定位**：专业、可信、有温度的 AI 健身伙伴

## 色彩

### 主色
- **KB Green** `#22C55E` — 活力、健康、成长
- 用于：主按钮、进度完成、正向反馈

### 辅助色
- **KB Dark** `#0F172A` — 深色背景、文字
- **KB Gray** `#64748B` — 次要文字、说明
- **KB Light** `#F8FAFC` — 浅色背景

### 功能色
- **警告橙** `#F59E0B` — 中等风险、需注意
- **危险红** `#EF4444` — 高风险、受伤警告
- **信息蓝** `#3B82F6` — 提示、链接

### 体态评分色
- 8-10 分：`#22C55E` 绿色
- 6-7 分：`#F59E0B` 黄色
- 1-5 分：`#EF4444` 红色

## 字体

- **标题**：PingFang SC / Noto Sans SC（中文）+ Inter（英文/数字）
- **正文**：PingFang SC / Noto Sans SC
- **数字/数据**：DIN Alternate / Inter（等宽，突出数据感）

### 字号
- H1：28px / Bold
- H2：22px / Bold
- H3：18px / Semibold
- Body：15px / Regular
- Caption：13px / Regular
- Small：11px / Regular

## 间距

基础单位：4px

- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px

## 圆角

- 小组件：8px
- 卡片：12px
- 按钮：12px
- 头像/照片：16px
- 全圆：999px

## 核心组件

### 体态雷达图
- 6 个维度：胸/肩/背/臂/腿/核心
- 中心为 0，边缘为 10
- 填充色：KB Green 30% 透明度
- 边框：KB Green

### 训练动作卡片
```

### 进度对比（滑块）
```
┌──────────────────────────────┐

### 训练计时器
- 大字体倒计时
- 震动提醒
- 可选音效
- 组间/组内两种模式

### 数据卡片
```
┌──────────────┐
│  体重         │
│  72.5 kg     │  ← 大号数字
│  ↓ 2.3kg     │  ← 变化（绿色下降/红色上升）
│  vs 上月      │
└──────────────┘
```

## 动效

- 页面切换：slide（300ms ease-out）
- 卡片出现：fade + slide up（200ms）
- 数据变化：数字滚动动画
- 完成打卡：confetti 庆祝效果
- 计时器：脉冲动画

## 空状态

- 未拍照："拍张照，开始你的蜕变之旅 📸"
- 无训练记录："今天开始第一次训练吧 💪"
- 无进度数据："坚持记录，见证变化 📊"

## 拍照引导界面

- 半透明人体轮廓线（辅助对齐）
- 角度检测提示（偏左/偏右/太近/太远）
- 三张标准照引导：正面 → 左侧 → 背面
- 拍摄完成动画反馈

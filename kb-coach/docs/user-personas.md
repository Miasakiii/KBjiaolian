# KB教练 — 目标用户画像

## Persona 1：小王 · 健身新手

- **年龄**：25 岁
- **职业**：程序员
- **健身经验**：0，刚办了健身房卡
- **痛点**：
  - 不知道怎么练，器械不会用
  - 怕练错受伤
  - 网上教程太多，不知道听谁的
  - 请私教太贵（月薪 8k，私教 300/次）
- **目标**：改善体态（久坐圆肩驼背），增肌变壮
- **使用场景**：
  - 拍照 → 看到自己圆肩 → AI 给出矫正方案
  - 跟着 App 练 → 动作演示 + 计时器
  - 每周拍照 → 看到变化 → 坚持动力

## Persona 2：小李 · 有基础遇瓶颈

- **年龄**：28 岁
- **职业**：设计师
- **健身经验**：2 年，断断续续
- **痛点**：
  - 训练没有系统性，想到什么练什么
  - 体重不变，身材也不变
  - 不知道哪里需要加强
  - 想要更专业的指导
- **目标**：突破瓶颈，身材更匀称
- **使用场景**：
  - 拍照分析 → 发现背部和手臂是短板
  - 生成针对性方案 → 重点加强薄弱部位
  - 8 周后对比 → 看到背部明显变宽

## Persona 3：小陈 · 体态问题困扰

- **年龄**：30 岁
- **职业**：行政
- **健身经验**：偶尔跑步
- **痛点**：
  - 久坐导致圆肩驼背严重
  - 腰疼，去医院检查说没什么大问题
  - 不知道做什么运动能改善
  - 瑜伽课太贵，跟视频又怕做错
- **目标**：改善体态，缓解腰背疼痛
- **使用场景**：
  - 拍照 → AI 识别出骨盆前倾 + 圆肩
  - 生成康复训练方案 → 低强度、安全
  - 4 周后对比 → 体态明显改善

## Persona 4：小赵 · 减脂塑形

- **年龄**：26 岁
- **职业**：销售
- **健身经验**：有，但减脂期总坚持不下来
- **痛点**：
  - 管不住嘴，节食反弹
  - 只看体重秤，不看体型变化
  - 缺乏可视化反馈
  - 没人监督
- **目标**：减脂 10kg，看到腹肌
- **使用场景**：
  - 拍照记录 → 看到脂肪分布
  - 每周拍照对比 → 虽然体重没变但腰围小了
  - 饮食拍照 → AI 估算热量
